9.表达式闪烁

  • 是因为angularjs没有加载完
  • 通过ng-bind 只能绑定单一的数据避免闪烁，且不能嵌套 (必须加载标签中 而且不能绑定多个)
  • 绑定多个 ng-bind-template=”“ 不能嵌套 标签内容为空
  • ng-cloak 防止闪烁 实现原理就是先隐藏标签（通过样式去隐藏），当angular 完成后再显示自动被移除ng-cloak
  • 此时所有数据都绑定在了根作用域下，没有实现模块化

14.ng-xxxx

- ng-model 双向绑定
- ng-disabled 禁用
- ng-if   要是条件不成立就不会渲染元素 一般多用于和ng-repeat 配合使用来判断数据是否存在  

- ng-show   显示  无论成立与否元素都会被渲染，一般的用于频繁的切换，第一次时候消耗比较大 只是简单的操作样式
- ng-hide   隐藏  无论成立与否元素都会被渲染，一般的用于频繁的切换，第一次时候消耗比较大 只是简单的操作样式

16.控制器的特点

- 创建controler时候会自动生成一个$scope ，把数据都挂在上$scope，实现双向的绑定（）
- 依赖注入  $scope,$sce
- DOM和控制器的作用范围是平行的
- 控制器可以嵌套，控制器所在dom标签可以嵌套
- 控制室不能服用的，而且作用是控制，（不要去操作dom） 唯一可以操作的地方是link函数

           var app = angular.module('appModule', []);
            app.directive('drag', function () {
                return {
                    //link 函数是唯一一个可以操作dom
                    //将视图和数据绑定（链接函数）    
                    //angular.element 将元素转化成jq对象
                    //scope 当前作用域   指令默认不会创建作用域
                    //element 当前指令的元素默认jq对象
                    //attr 元素属性
                    link: function (scope, element, attrs) {
                        element.on('mousedown', function (e) {
                            var disX = e.pageX - this.offsetLeft;
                            var disY = e.pageY - this.offsetTop;
                            angular.element(document).on('mousemove', function (e) {
                                var curX = e.pageX - disX;
                                var curY = e.pageY - disY;
                                //在angular 中使用必须要增加单位
                                element.css({left: curX + "px", top: curY + "px"});
                            });
                            angular.element(document).on('mouseup', function () {
                                angular.element(document).off();
                            })
                            e.preventDefault();
                        });    
                    }    
                }            
            });
